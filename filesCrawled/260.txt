<h2><span class="mw-headline" id="Surnames">Surnames</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Georgian_name&amp;action=edit&amp;section=2" title="Edit section: Surnames">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/File:CoA_of_the_Most_Serene_Prince_Dadian_of_Mingrelia.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b5/CoA_of_the_Most_Serene_Prince_Dadian_of_Mingrelia.png/220px-CoA_of_the_Most_Serene_Prince_Dadian_of_Mingrelia.png" decoding="async" width="220" height="258" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b5/CoA_of_the_Most_Serene_Prince_Dadian_of_Mingrelia.png/330px-CoA_of_the_Most_Serene_Prince_Dadian_of_Mingrelia.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b5/CoA_of_the_Most_Serene_Prince_Dadian_of_Mingrelia.png/440px-CoA_of_the_Most_Serene_Prince_Dadian_of_Mingrelia.png 2x" data-file-width="682" data-file-height="799" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/File:CoA_of_the_Most_Serene_Prince_Dadian_of_Mingrelia.png" class="internal" title="Enlarge"></a></div>Coat of arms of the princely family of <a href="/wiki/House_of_Chikovani" title="House of Chikovani">Chikovani</a>.</div></div></div>
<p>Georgian surnames are derived either from <a href="/wiki/Patronymic" title="Patronymic">patronymics</a> or, less frequently, from <a href="/wiki/Toponym" class="mw-redirect" title="Toponym">toponyms</a>, with addition of various suffixes.
</p><p>Georgian suffixes vary by region. The most common Georgian suffixes are:
</p>
<ul><li><b>-shvili</b> (-შვილი): meaning "<a href="/wiki/Child" title="Child">child</a>": from western and eastern <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Baratashvili" title="Baratashvili">Baratashvili</a>, <a href="/wiki/Andronikashvili" title="Andronikashvili">Andronikashvili</a>, <a href="/wiki/Guramishvili" title="Guramishvili">Guramishvili</a>, <a href="/wiki/Jughashvili" class="mw-redirect" title="Jughashvili">Jughashvili</a> etc.</li>
<li><b>-dze</b> (-ძე): meaning "<a href="/wiki/Son" title="Son">son</a>": from western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Abashidze" title="Abashidze">Abashidze</a>, <a href="/wiki/Chavchavadze" title="Chavchavadze">Chavchavadze</a> etc.</li>
<li><b>-eli</b> (-ელი): meaning "<a href="/wiki/Preposition_and_postposition" title="Preposition and postposition">from</a> (place)": from eastern and western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Jaqeli" class="mw-redirect" title="Jaqeli">Jaqeli</a>, <a href="/wiki/Tsereteli" title="Tsereteli">Tsereteli</a>, <a href="/wiki/Amashukeli" title="Amashukeli">Amashukeli</a> etc.</li>
<li><b>-uri</b> and <b>-uli</b> (-ური) and (-ული): from mountainous eastern <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Donauri" title="Donauri">Donauri</a> and <a href="/wiki/Burduli" class="mw-redirect" title="Burduli">Burduli</a></li>
<li><b>-ani</b> (-ანი): <a href="/wiki/Svans" title="Svans">Svan</a> surname from mountainous western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Dadeshkeliani" class="mw-redirect" title="Dadeshkeliani">Dadeshkeliani</a>, <a href="/wiki/Dadiani" class="mw-redirect" title="Dadiani">Dadiani</a>, <a href="/wiki/Kipiani" title="Kipiani">Kipiani</a> etc.</li>
<li><b>-ia</b>, <b>-ua</b>, <b>-ava</b> and <b>-skiri</b> (-ია), (-უა), (-ავა) and (-სკირი): <a href="/wiki/Mingrelians" title="Mingrelians">Mingrelian</a> surname from western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Abakelia" title="Abakelia">Abakelia</a>, <a href="/wiki/Chichua" class="mw-redirect" title="Chichua">Chichua</a>, <a href="/wiki/Gvazava" title="Gvazava">Gvazava</a>, <a href="/wiki/Irakli_Kvekveskiri" title="Irakli Kvekveskiri">Kvekveskiri</a> etc.</li>
<li><b>-shi</b> (-ში): <a href="/wiki/Laz_people" title="Laz people">Laz</a> surname from western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Tugushi" title="Tugushi">Tugushi</a>, <a href="/wiki/Sopho_Khalvashi" title="Sopho Khalvashi">Khalvashi</a> etc.</li>
<li><b>-khi</b> (-ხი): from southern <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Meskhi" title="Meskhi">Meskhi</a>, <a href="/wiki/Revaz_Lashkhi" title="Revaz Lashkhi">Lashkhi</a> etc.</li>
<li><b>-kva</b> (-ყვა): from western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Pavle_Ingorokva" title="Pavle Ingorokva">Ingorokva</a>, Rokva etc.</li>
<li><b>-ioni</b> (-იონი): from eastern and western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Bagrationi" class="mw-redirect" title="Bagrationi">Bagrationi</a>, Kimerioni etc.</li>
<li><b>-ti</b> (-ტი): from western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. <a href="/wiki/Glonti" title="Glonti">Glonti</a>, <a href="/wiki/Giorgi_Jgenti" title="Giorgi Jgenti">Jgenti</a> etc.</li>
<li><b>-di</b> (-დი): surname from mountainous eastern <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a> E.g. Dzolbordi.</li>
<li><b>-ba</b> (-ბა): from western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g. Ukleba.</li>
<li><b>-re</b> (-რე): from western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g.Galogre</li>
<li><b>-ri</b> (-რი): from western <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>. E.g Mudziri.</li></ul>
<p>The first recorded Georgian surnames date to the 7th–8th century. They were mostly toponymic in nature (such as <a href="/wiki/Surameli" title="Surameli">Surameli</a>, <a href="/wiki/Machabeli" title="Machabeli">Machabeli</a>, <a href="/wiki/House_of_Orbeli" title="House of Orbeli">Orbeli</a>), patronymic, or derived from the profession, social status, position, or title, which was hereditary in the family (such as <a href="/wiki/Amilakhvari" title="Amilakhvari">Amilakhvari</a>, <a href="/wiki/Amirejibi" title="Amirejibi">Amirejibi</a>, 
<a href="/wiki/Eristavi" title="Eristavi">Eristavi</a> etc.). Beginning from the 13th century, the surnames became more frequently based upon patronymics, a tradition which became almost universal in the 17th–18th century. Some of the Georgian surnames indicate ethnicity or regional origin of the family, and are also generated as patronymics. Examples are Kartvelishvili ("child of Kartveli", i.e., <a href="/wiki/Georgians" title="Georgians">Georgian</a>), Megrelishvili ("child of Megreli", i.e., Mingrelian), <a href="/wiki/Cherkezishvili" title="Cherkezishvili">Cherkezishvili</a> ("child of Cherkezi", i.e., <a href="/wiki/Circassians" title="Circassians">Circassian</a>), <a href="/wiki/Abkhazi" title="Abkhazi">Abkhazishvili</a> ("child of Abkhazi", i.e., <a href="/wiki/Abkhazia" title="Abkhazia">Abkhazian</a>), Somkhishvili ("child of Somekhi", i.e., <a href="/wiki/Armenians" title="Armenians">Armenian</a>), Berdzenishvili ("child of Berdzeni", i.e., <a href="/wiki/Greeks" title="Greeks">Greek</a>), Prangishvili ("child of Prangi", i.e., <a href="/wiki/French_people" title="French people">French</a>).    
</p><p>There are some very rare Georgian surnames like Jolbordi, Galogre, Lapachi, Molodini, Shermadini, Sivsive etc.
